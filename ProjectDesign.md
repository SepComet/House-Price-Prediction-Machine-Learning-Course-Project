# 房价预测机器学习课程项目规划

## 一、项目名称

**基于机器学习算法的房价预测研究**

---

## 二、项目目标

本项目拟使用机器学习方法，根据房屋相关特征数据建立房价预测模型，对房屋价格进行回归预测，并比较不同算法在该任务中的表现差异，分析影响房价预测效果的关键因素。

---

## 三、问题描述

房价受到多种因素影响，例如地区收入水平、房龄、房屋平均房间数、人口密度以及地理位置等。  
传统方式往往难以准确刻画这些复杂关系，因此本项目希望通过机器学习算法，从已有房屋数据中自动学习特征与房价之间的映射关系，从而实现对未知样本房价的预测。

该问题属于：

- **监督学习（Supervised Learning）**
- **回归问题（Regression Problem）**

---

## 四、数据来源

### 1. 数据集选择

本项目选用 **California Housing（加州房价数据集）** 作为实验数据集。

### 2. 数据来源方式

该数据集可通过 Python 的 `scikit-learn` 库直接获取，无需手动下载，使用方便，适合作为课程实验数据。

示例获取方式：

```python
from sklearn.datasets import fetch_california_housing
housing = fetch_california_housing()
```

### 3. 数据集内容简介

该数据集包含多个与房价相关的特征，常见特征包括：

* `MedInc`：地区住户收入中位数
* `HouseAge`：房龄中位数
* `AveRooms`：平均房间数
* `AveBedrms`：平均卧室数
* `Population`：地区人口数
* `AveOccup`：平均居住人数
* `Latitude`：纬度
* `Longitude`：经度

预测目标为：

* `MedHouseVal`：地区房价中位数

### 4. 选择该数据集的原因

* 数据公开、规范，便于直接使用
* 特征与目标明确，适合回归建模
* 数据质量较高，减少前期清洗难度
* 适合进行多种机器学习算法对比实验
* 便于完成课程报告中的实验分析与可视化展示

## 八、拟使用的机器学习算法

为了比较不同算法在房价预测任务中的表现，本项目计划采用以下回归算法：

### 1. 线性回归（Linear Regression）

#### 作用

作为基础模型，用于建立房屋特征与房价之间的线性关系。

#### 选择原因

* 原理简单，易于理解
* 适合作为 baseline 模型
* 便于与其他复杂模型进行比较

---

### 2. 决策树回归（Decision Tree Regression）

#### 作用

利用树结构学习特征与房价之间的非线性关系。

#### 选择原因

* 能处理较复杂的特征关系
* 结果相对直观，易于解释
* 可反映非线性建模能力

---

### 3. 随机森林回归（Random Forest Regression）

#### 作用

通过集成多棵决策树，提高预测稳定性和准确性。

#### 选择原因

* 对非线性问题有较好的拟合能力
* 相比单棵决策树更稳定
* 在回归任务中通常表现较好

---

### 4. 可选扩展算法（视时间情况决定是否加入）

如果后续时间允许，可扩展以下算法：

## 九、模型评估指标

由于本项目属于回归问题，计划采用以下评估指标：

### 1. MAE（平均绝对误差）

用于衡量预测值与真实值之间平均偏差的绝对大小。

### 2. MSE（均方误差）

用于衡量预测误差平方后的平均值，对大误差更敏感。

### 3. RMSE（均方根误差）

MSE 开方后的结果，量纲与房价一致，更便于理解。

### 4. R²（决定系数）

用于衡量模型对数据的解释能力，数值越接近 1 表示拟合效果越好。
